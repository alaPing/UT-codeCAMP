## カート機能仕様

### カートに入れる

カートに商品を 1 個入れることができます。

### カートから取り出す

カートから商品を取り出すことができます。
個数は問いません。

### カートの中を見る

カートの中に入った商品と個数の組を見ることができます。

### 商品の個数を変更する

商品 ID を指定して、個数を変更することができます。
個数は負でない整数である必要があります。
0 を指定した場合、カートから削除されます。

### 合計金額を計算する

商品の合計金額を得ることができます。
