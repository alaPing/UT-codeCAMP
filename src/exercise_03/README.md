### ダメージ計算関数
攻撃者のLv        整数型 1~100  
威力              整数型 1~  
攻撃者のこうげき  整数型 1~  
防御者のぼうぎょ  整数型 1~  
  
ダメージ = [[[攻撃者のLv * 0.4 + 2] * 威力 * 攻撃者のこうげき / 防御者のぼうぎょ] / 50 + 2]
